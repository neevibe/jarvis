This is what one long-term memory looks like: a single markdown file you can
open, edit, or delete by hand. The files are the source of truth; `index.md`
is generated from them.

**The fact.** State what happened or what is true, with its context: who
decided, when, and the alternatives that were rejected.

**Why it matters.** The reasoning — this is what makes the memory useful
months later, when the fact alone would be ambiguous.

**How to apply it.** What the agent should do differently because it knows
this.

Valid types: identity-fact, preference, decision, project, person, reference.
The `hook` is the one-line searchable summary shown in the index.
